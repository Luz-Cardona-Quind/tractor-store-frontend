# Skill: Task Analysis (MFE)

**Scope**: PlanningMF📋  
**Trigger**: "crear MFE", "implementar feature", "configurar routing"  
**Hand-off**: → implementation-plan

---

## Propósito

Analizar solicitudes del usuario y descomponer en tareas específicas, ordenadas y con dependencias claras para Micro Frontends.

---

## Proceso

1. Identificar tipo de tarea (MFE nuevo, componente, routing, estilos, Figma)
2. Listar subtareas necesarias
3. Determinar dependencias entre subtareas
4. Establecer orden de ejecución
5. Identificar agentes especializados para cada subtarea
